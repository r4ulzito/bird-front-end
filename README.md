# Como executar:
```
# Clone o projeto
git clone https://gitlab.com/equipe-rocket-2022/front-end.git
# Acesse
cd front-end
# Instale as dependecias
yarn add or npm install
# Inicie o projeto
yarn dev or npm run dev
```
